Plugins

Survey plugin development

Custom plugins that extend what Qualtrics and QuestionPro can capture, built and deployed inside the platform itself.

  • Voice Response Capture for QuestionPro, a browser based audio recorder plugin letting respondents record 20 second voice responses without leaving the survey
  • Built on the MediaRecorder API with intelligent chunking, audio stored securely inside the survey platform, no file uploads or external services
  • Backend processing engine extracts, reconstructs, and converts audio into analysis ready MP3 files, organized by respondent
  • Deployed at scale, proven across 300+ live survey respondents
Built inside the platform, no added infrastructure
